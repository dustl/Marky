# Marky

> 简洁而优雅的 Markdown 编辑器 

Marky 是一款为写作而生的 Markdown 编辑器。它融合了所见即所得的流畅体验与纯文本的极致控制力，让创作者专注于内容本身，而非工具。

---

## 特性

### 书写即呈现

Marky 采用块级渲染引擎，每一次按键都实时解析 Markdown 语法，将标题、列表、代码块、引用等元素即刻呈现在眼前。你在书写，也在预览。

### 数学与图表

支持 KaTeX 行内/块级数学公式（`$...$` 与 `$$...$$`），以及 Mermaid 流程图、时序图、甘特图等图表渲染。技术写作，浑然一体。

### 代码高亮

内置 highlight.js，支持数十种编程语言的语法着色。代码块自动标注语言标签、显示行号，并提供一键复制按钮，让技术文档的阅读与分享同样愉悦。

### 智能编辑

- **自动配对** — 输入引号或括号时自动补全，选中文本时自动包裹

- **Markdown 快捷输入** — `>`、`-`、`#`、`1.` 等标记后接空格，立即转换为对应格式

- **智能列表** — 有序列表、无序列表、任务列表自动延续，`---` 回车快速插入分割线

- **搜索替换** — 全文档搜索与批量替换，精准高效

### 写作体验

- **专注模式** — 隐去工具栏与状态栏，沉浸于纯粹的书写空间

- **打字机模式** — 当前行始终保持在屏幕中央，模拟打字机的专注节奏

- **全屏写作** — 一键全屏，整个世界只剩下你和文字

- **当前行高亮** — 细微的光标行背景提示，让你永不迷失行踪

### 图片处理

支持粘贴剪切板图片，自动保存至 `Marky_assets` 目录。自定义语法 `![alt](url =WxH)` 可精确控制图片显示尺寸。

### 拖拽交互

- **块拖拽排序** — 拖拽块左侧手柄（⠿）随意调整文档结构

- **拖拽打开文件** — 将 `.md` 文件拖入窗口即可打开

### 模板与表格

内置 README、Changelog、Meeting Notes、Blog Post 四种文档模板，一键插入。表格插入采用直观的 10×10 网格选取器，所见即所得。

### 导航与统计

- **大纲面板** — 侧边栏展示文档标题结构，点击快速跳转

- **字数详情** — 点击状态栏字数，弹窗展示词汇数、字符数（含/不含空格）、段落数、句子数及预估阅读时间

- **阅读时间估算** — 状态栏实时显示 "X min read"

### 自动保存

60 秒间隔自动保存，手动保存自动重置计时器。不再担心意外丢失。

### 快捷键

| 快捷键 | 功能 |
| --- | --- |
| `Cmd+N` | 新建文档 |
| `Cmd+O` | 打开文件 |
| `Cmd+S` | 保存 |
| `Cmd+Shift+S` | 另存为 |
| `Cmd+E` | 导出 HTML |
| `Cmd+Z` / `Cmd+Shift+Z` | 撤销 / 重做 |
| `Cmd+F` | 搜索替换 |
| `Cmd+/` | 快捷键参考 |
| `Cmd+Shift+T` | 切换主题 |
| `Cmd+Shift+O` | 切换大纲 |
| `Cmd+Shift+F` | 专注模式 |
| `Cmd+Shift+W` | 打字机模式 |
| `Cmd+Shift+.` | 全屏 |
| `Tab` | 缩进（2 空格） |

按 `Cmd+/` 可在编辑器中随时查看完整的快捷键列表。

### 最近文件

自动记录最近打开的 10 个文件，置于 File → Open Recent 菜单，亦可从 Finder 通过「打开方式」直接打开。

### 窗口记忆

窗口位置与尺寸自动持久化，每次打开都回到你熟悉的位置。

## 开始使用

```bash
# 安装依赖
npm install

# 启动开发模式（含 DevTools）
npm run dev

# 启动应用
npm start

# 构建发行包
npm run build
```

## 技术栈

- **Electron** — 跨平台桌面应用框架

- **marked** — Markdown 解析与渲染

- **highlight.js** — 代码语法高亮

- **KaTeX** — 数学公式渲染

- **Mermaid** — 图表与流程图渲染
